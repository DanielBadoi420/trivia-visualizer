# Open Trivia Visualizer
A simple React app that fetches trivia questions from the [Open Trivia DB API](https://opentdb.com/) and visualizes them by **category** and **difficulty** using [Recharts](https://recharts.org/).
## Features
- Fetches 50+ trivia questions from the Open Trivia DB API  
- Displays:
  - Distribution of questions by **category** (bar chart)  
  - Distribution of questions by **difficulty** (pie chart)  
- Allows filtering by category  
## Tech
- React 
- Recharts    
- GitHub Pages
## Setup & Run Locally
# Clone the repository
git clone https://github.com/DanielBadoi420/trivia-visualizer.git

# Go into the folder
cd trivia-visualizer

# Install dependencies
npm install

# Start the app
npm start
